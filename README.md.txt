# 🛡️ FraudGuard AI: Enterprise Detection System

A high-performance machine learning dashboard designed to detect fraudulent financial transactions in real-time.

## 🚀 Features
- **Random Forest Classifier:** Trained on 6M+ transaction logs.
- **Real-time Risk Gauges:** Visualizes fraud probability instantly.
- **Enterprise UI:** Glassmorphism design with professional metrics.
- **Explainable AI (XAI):** Breakdown of factors contributing to risk scores.

## 🛠️ Tech Stack
- **Language:** Python 3.11
- **ML Library:** Scikit-Learn / Joblib
- **Frontend:** Streamlit
- **Visualization:** Plotly / Pandas

## 📦 Installation
1. Clone the repo
2. Install requirements: `pip install -r requirements.txt`
3. Run: `streamlit run app.py`
